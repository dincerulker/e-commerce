# e-commerce

## All necessary installations should be done as follows.

1. Install nodejs from https://nodejs.org/en/download/
2. Open terminal or command
3. run "npm install -g"
4. run "npm init"
5. run "npm install bootstrap"
6. run "npm i js-cookie" for cookie package
